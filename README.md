# CRUD com React Router DOM 🚀

Este projeto foi desenvolvido para a disciplina de Desenvolvimento Web no **IFMG**. O objetivo principal é a implementação de um sistema de navegação dinâmica (SPA) em uma aplicação CRUD, utilizando as melhores práticas de roteamento do React.

## 🎯 Objetivos da Atividade
- [x] Criar um **Menu Fixo** de navegação.
- [x] Configurar as rotas da aplicação utilizando `react-router-dom`.
- [x] Implementar navegação entre as páginas sem recarregamento (SPA).
- [x] Utilizar recursos avançados como `NavLink` e `useNavigate`.

## 🛠️ Tecnologias e Recursos
- **React.js** (Vite)
- **React Router DOM v6**: 
    - `BrowserRouter`, `Routes` e `Route` para gestão de caminhos.
    - `NavLink` para links de navegação com feedback visual de estado ativo.
    - `useNavigate` para redirecionamento programático após ações (como salvar um item).
- **CSS3 Moderno**: Design com foco em UX, utilizando fontes personalizadas e layout responsivo.

## 📂 Estrutura do Projeto
A organização segue o padrão de separação de responsabilidades:
- `src/components/`: Componentes globais (Navbar).
- `src/pages/`: Telas da aplicação (Home e Cadastro).
- `src/App.jsx`: Definição e gestão das rotas.
